  .claude/ と .cascade-settings.json を除外した理由

  - 個人の作業環境設定（エディタや AI
  ツールの設定）はマシンやユーザーごとに違うから、リポジトリで共有する意味が薄い
  - 他の人が clone したとき、自分の環境と競合する可能性がある
  - うっかり API キーや認証情報が含まれてるケースもあるから安全側に倒すのが無難

  逆にコミットすべき設定ファイルの例

  ┌────────────────────────────┬──────────────────────────────────────┐
  │          ファイル          │                 理由                 │
  ├────────────────────────────┼──────────────────────────────────────┤
  │ .editorconfig              │ チーム全員で統一したいコードスタイル │
  ├────────────────────────────┼──────────────────────────────────────┤
  │ pyproject.toml / setup.cfg │ プロジェクトのビルド・lint設定       │
  ├────────────────────────────┼──────────────────────────────────────┤
  │ requirements.txt           │ 依存パッケージ（再現性のため必須）   │
  ├────────────────────────────┼──────────────────────────────────────┤
  │ .github/workflows/         │ CI/CD パイプライン                   │
  └────────────────────────────┴──────────────────────────────────────┘
