## 実施内容

### 1. 検証フレームワーク設計

- 3アルゴリズムの予測精度を継続的に検証するフローを設計
- **データ更新 → 予測実行 → 抽選まで待機 → 答え合わせ → 蓄積・分析** の5ステップサイクル
- 検証期間: 無期限（蓄積データが増えるほど精度評価が信頼できる）

### 2. `src/verification/update_data.py` — CSVデータ自動更新

- `loto-life.net` のCSV配信URL（`/csv/loto6`, `/csv/loto7`, `/csv/mini`）からShift-JISデータを取得
- 既存CSVとの差分マージ（開催回番号で重複排除・昇順ソート）
- 標準ライブラリのみ（`urllib.request`, `csv`, `io`）で実装。外部依存なし
- `--dry-run` オプションで動作確認のみ可能
- **初回実行結果**: ロト6 +6件、ロト7 +2件、ミニロト +3件を取得

```powershell
# 全ゲームを更新（通常はこれだけ）
python -m src.verification.update_data

# 特定ゲームのみ
python -m src.verification.update_data --game loto6

# 動作確認のみ（保存しない）
python -m src.verification.update_data --dry-run
```

### 3. `src/verification/run_prediction.py` — 一括予測実行

- 3ゲーム × 3手法 = 最大9パターンの予測を一括実行
- 各アルゴリズムのAPI呼び出し調整:
  - **モンテカルロ**: `simulator.run()` → `analyze_top_combinations()` でトップ10件取得
  - **クラスタリング**: `generate_predictions()` はタプルのリストを返すので `sorted(p)` で処理
  - **GA**: `GAResult.best_individuals` はタプルのリストなので `sorted(ind)` で処理
- 結果を `data/predictions/prediction_{game}_{YYYYMMDD}.json` に保存
- 予測数: モンテカルロ=10セット、クラスタリング=5セット、GA=5セット

```powershell
# 全ゲーム全手法（通常はこれだけ）
python -m src.verification.run_prediction

# 特定ゲームのみ
python -m src.verification.run_prediction --game loto6

# 特定手法のみ
python -m src.verification.run_prediction --method montecarlo genetic
```

### 4. `src/verification/verify_results.py` — 答え合わせ自動化

- 保存済み予測JSONの最新ファイルを自動選択（ゲームキーでフィルタリング）
- CSVから当選回号を自動取得（`--draw` で回号指定、省略時は最新予測の次回を自動推定）
- ロト6/7/ミニロトの当選等級判定を全等級実装:

| ゲーム | 最高等級 | 判定基準 |
|--------|---------|---------|
| ロト6 | 1等〜5等 | 本数字6個一致〜3個一致 |
| ロト7 | 1等〜7等 | 本数字7個一致〜本数字3個+ボーナス1個 |
| ミニロト | 1等〜4等 | 本数字5個一致〜3個一致 |

- 照合結果を `data/results/verify_{game}_draw{回号}_{予測日}.json` に保存
- 手動入力オプション（CSVに結果が反映される前でも照合可能）

```powershell
# CSVから自動取得して照合（update_data後に実行）
python -m src.verification.verify_results

# 当選番号を手動入力（CSVに反映前でも照合可能）
python -m src.verification.verify_results --game loto6 --draw 2083 --winning 5 12 20 31 37 42 --bonus 8

# 特定の予測ファイルを指定
python -m src.verification.verify_results --file data/predictions/prediction_loto6_20260305.json
```
