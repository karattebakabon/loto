# 🎰 Loto Predictor — ロト予測ツール

> 買わなきゃ当たらない。でも、買うなら賢く買いたい。

ロト6・ロト7・ミニロトの当選番号を、統計とAIの力でちょっとだけ有利に予測してみよう！…というPythonツール群です。

## 🤔 これ何？

Geminiとのブレストで「ロトの数字って何かしらの傾向あるんじゃない？」という話になり、じゃあ実際にコード書いて試してみようぜ！と始まったプロジェクトです。

**当たるかどうかは知りません。** でも作る過程が楽しければそれでいいじゃない？

## ✨ 特徴

- **🎲 マルチゲーム対応** — ロト6 / ロト7 / ミニロト、設定一発切り替え
- **🧠 複数アルゴリズム** — 違うアプローチで予測して比較できる
- **⚡ ボーナス数字も考慮** — 本数字とボーナスの重みを分けて分析

## 📊 搭載予定のアルゴリズム

| アルゴリズム | ひとことで言うと | ステータス |
|-------------|-----------------|-----------|
| モンテカルロ・シミュレーション | 「100万回仮想抽選して、よく出るパターンを探す」 | 🔜 これから |
| クラスタリング（K-Means / DBSCAN） | 「過去データを地図にして、当たりの密集地帯を見つける」 | 🔜 これから |
| 遺伝的アルゴリズム（GA） | 「数字の組み合わせを進化させて最強個体を育てる」 | 🔜 これから |

## 🎯 対応ゲーム

| ゲーム | 数字範囲 | 選択数 | ボーナス数字 |
|--------|---------|--------|-------------|
| ロト6 | 1〜43 | 6個 | 1個 |
| ロト7 | 1〜37 | 7個 | 2個 |
| ミニロト | 1〜31 | 5個 | 1個 |

## 🚀 セットアップ

```bash
# クローン
git clone https://github.com/<your-username>/loto.git
cd loto

# 仮想環境つくる
python -m venv .venv
.venv\Scripts\activate   # Windows

# パッケージ入れる
pip install -r requirements.txt
```
